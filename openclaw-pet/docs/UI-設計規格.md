## 5. 龍蝦角色規格

### 5.1 尺寸選項

| 尺寸 | 像素 | 用途 |

| 小 | 32x32 | 隱藏模式 |

|------|-----|

| 驚訝 | O O | O | 舉起 |
| 睡覺 | .. | ~ | 靜止 |
| 說話 | ^ ^ | 張合 | 擺動 |

     🦞

| 狀態 | 背景 | 圖示顏色 |
|------|------|----------|
| 正常 | 透明 | #636E72 |

## 7. 動畫規格

### 7.1 過場動畫

| 動畫 | 時長 | 曲線 |
|------|------|------|
| 氣泡出現 | 200ms | easeOut |
| 氣泡消失 | 150ms | easeIn |
| 視窗開啟 | 300ms | spring |
| 視窗關閉 | 200ms | easeIn |

### 7.2 龍蝦動作

| 動作 | 描述 | 循環 |
|------|------|------|
| idle | 左右走動 + 觸手擺動 | 是 |
| talking | 說話時點頭 + 觸手動 | 是 |
| thinking | 轉圈思考 | 是 |
| happy | 跳躍 + 愛心效果 | 否 |
| sleeping | 打呼 (Zzz 漂浮) | 是 |
| alert | 震動 + 驚嘆號 | 否 |
